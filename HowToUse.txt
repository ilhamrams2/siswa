# E-Learning Application Starter

Use Codeigniter 4 and PHP 7 !!!

1. melakukan clone, diharuskan membuka terminal untuk melakukan "composer update" di dalam folder yang di clone
2. Buka PhpMyadmin untuk membuat database "db_elearning" setelah jadi pergi ke halaman import lalu pilih file db_elearning.sql dan import
3. Ubah config database pergi ke app/config/database.php setelah itu sesuaikan username / password  database anda
4. tuliskan perintah di terminal php spark serve untuk menjalankan server
